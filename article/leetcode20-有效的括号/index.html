leetcode20-有效的括号

2019-12-11
  • 算法/数据结构

原题

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

示例1:

输入: “()”
输出: true

示例2:

输入: “()[]{}”
输出: true

示例3:

输入: “(]”
输出: false

示例4:

输入: “([)]”
输出: false

示例5:

输入: “{[]}”
输出: true

解法

思想

左括号全部入栈，如果是右括号寻找栈首是否为对应的左括号，存在就将左括号出栈，直到栈元素全部清空。
